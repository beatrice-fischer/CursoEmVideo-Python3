# 🐍 Curso em Vídeo — Python 3
- Este repositório reúne os exercícios resolvidos durante o Curso em Vídeo de Python, ministrado pelo professor Gustavo Guanabara.
- O objetivo deste projeto é registrar meu processo de aprendizado em Python por meio da prática, servindo como referência futura e como base para projetos mais complexos.
- Cada exercício representa uma etapa da evolução na linguagem — desde os primeiros conceitos até estruturas de dados e organização de código.

## 📚 Estrutura do curso
O curso é dividido em três módulos principais, chamados de Mundos:

## 🌍 Mundo 1 — Fundamentos
- Exercícios abordando os conceitos iniciais da linguagem:
  - Entrada e saída de dados, operadores aritméticos, manipulação de strings.
  - Condicionais básicas, primeiros laços de repetição.
- Essa etapa estabelece a base da lógica de programação em Python.


### 🌍 Mundo 2 — Estruturas de Controle
- Exercícios aprofundando o controle do fluxo do programa:
  - Estruturas if, elif e else.
